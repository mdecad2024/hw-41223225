python  main.py

之後再複製init_replit中的

pip install flask flask_cors bs4 lxml pelican markdown gevent

就可以有連結進入此網站開啟編輯

近端:先去下載http://229.cycu.org/python_2025_lite.7z到USB

在進入ipv6

接著就是cd tmp  

tmp>git clone https://github.com/mdecad2024/hw-412232XX.git (此為進入你的網站並且複製下來)

接著就可以再回到tmp 此時就可cd hw-412232XX.git然後cms執行

接著進入複製的網站就可以開始編輯

切記每次要用的時候都在回去github網站copy一次避免檔案沒有下載好git pull

或者如果沒有傳上去記得git push
